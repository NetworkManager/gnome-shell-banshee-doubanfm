# DoubanFM Banshee Gnome Shell Extension #

# Dependency #
* Gnome Shell >= 3.6
* Banshee >= 2.4
* Banshee Douban FM plugin >= 0.3.4 (https://bitbucket.org/pro711/banshee-doubanfm/wiki/Home)

# Usage #
Use mouse click on the extension tab show on the panel
* Left: Next Song
* Middle: Hate Song
* Right: Love (toggle)
